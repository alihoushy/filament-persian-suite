# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- In-house Jalali date conversion (no external dependencies)
- JalaliDatePicker form component with Alpine.js integration
- JalaliDateColumn table column with formatting options
- JalaliDateRangeFilter for filtering by date range
- PersianNumberFormatter for converting English to Persian digits
- FormatsPersianNumbers trait for easy integration
- Vazirmatn font integration with RTL support
- Tailwind v4 compatible styling with dark mode support
- Comprehensive Pest test coverage
- Full documentation in Persian and English

### Technical Details
- PHP 8.3+ support
- Laravel 11.0+ (LTS) compatibility
- Filament v4 architecture compliance
- PSR-4 autoloading
